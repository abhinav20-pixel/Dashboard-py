# Dash Dashboard Project

This project is built using Python and Dash.

Follow the steps below to run the project on your system.

---

## 1. Install Python

Download and install Python from:
https://www.python.org/downloads/

During installation, enable:

* Add Python to PATH
* pip
* venv

---

## 2. Clone the repository

```
git clone https://github.com/YOUR_USERNAME/dash-dashboard.git
cd dash-dashboard
```

---

## 3. Create virtual environment

```
python -m venv .venv
```

---

## 4. Activate virtual environment

### Windows:

```
.venv\Scripts\activate
```

### Mac/Linux:

```
source .venv/bin/activate
```

---

## 5. Install required libraries

```
pip install -r requirements.txt
```

---

## 6. Run the Dash application

```
python main.py
```

---

## 7. Open in browser

```
http://127.0.0.1:8050/
```

The dashboard will start locally.
